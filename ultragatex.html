Under The Hood

Clean Plugin Architecture

Every feature lives in its own file. Adding a new feature means creating one new .py file — nothing else changes.

UltraGateX/
├── main.py             # Entry point — loads plugins
├── config.py         # Env vars & constants
│
├── core/              # Shared utilities
│   ├── security.py    # Auth, key gen, QR
│   ├── analytics.py   # Events, fraud, geo
│   └── github_sync.py # Push licenses.json
│
├── plugins/        # 🔌 One file = one feature
│   ├── licenses.py    # /genkey /editkey
│   ├── honeypot.py    # Trap system
│   └── updater.py     # /update self-update
│
└── data/             # JSON data (gitignored)
