Introduction

Welcome to the JusticeTech System API documentation. Our REST APIs let you integrate JMWhatsApp, Miss Chatra, and Contact Autosave directly into your applications.

All API requests require authentication via API key, and responses are returned in JSON format with consistent error handling across all three products.

Base URL

BASE https://api.justicetechsystem.space/v1

Authentication

Include your API key in the request header:

# Add to every request header
Authorization: Bearer YOUR_API_KEY

JMWhatsApp — Send Message

Send a WhatsApp message programmatically through your connected JMWhatsApp instance.

POST /jmwhatsapp/send
ParameterTypeDescription
tostringRecipient phone number (with country code)
messagestringText content to send
media_urlstring?Optional media attachment URL
// Example request
{
  "to": "+2348012345678",
  "message": "Hello from the API!"
}

Miss Chatra — Chat Completion

Send a message to Miss Chatra's AI engine and receive a context-aware response routed to the optimal AI endpoint.

POST /chatra/complete
ParameterTypeDescription
promptstringThe user's message
session_idstringConversation ID for context retention
endpointstring?Force a specific AI model (optional)
// Example response
{
  "reply": "Sure, I can help with that!",
  "endpoint_used": "gemini-2.5-flash",
  "latency_ms": 340
}

Contact Autosave — List Contacts

Retrieve all synced contacts with pagination support.

GET /autosave/contacts?page=1&limit=50
ParameterTypeDescription
pageintegerPage number (default: 1)
limitintegerResults per page (max: 100)
# cURL example
curl https://api.justicetechsystem.space/v1/autosave/contacts \
  -H "Authorization: Bearer YOUR_KEY"
